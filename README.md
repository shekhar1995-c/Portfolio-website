## Setup

1. Clone or download this repository
2. Open `index.html` in your browser
3. Customize with your information (see instructions in `index.html`)

## Customization

### Add Your Photo
1. Create an `assets` folder
2. Add your photo as `assets/profile.jpg`
3. Uncomment the img tag in the About section

### Update Content
Look for `<!-- INSTRUCTION: ... -->` comments in `index.html` to find sections that need your personal information:
- Bio and contact details
- Work experience
- Skills
- Projects
- Social media links

### Contact Form
To make the contact form functional, integrate with:
- [EmailJS](https://www.emailjs.com/)
- [Formspree](https://formspree.io/)
- Your own backend API
